2. FORMATO DE DATOS
-------------------
Los datos extraídos de la PokeAPI están en formato JSON (JavaScript Object Notation), 
que es un formato de datos SEMI-ESTRUCTURADOS.

¿Por qué semi-estructurados?

1. Estructura definida: Cada pokémon tiene una estructura consistente con campos 
   específicos como name, types, stats, height, weight, etc.

2. Formato estandarizado: Los datos siguen un esquema predefinido de la API, 
   donde cada pokémon tiene los mismos tipos de atributos.

3. No completamente estructurados: Aunque tienen estructura, no están organizados 
   en una tabla relacional tradicional como un CSV o base de datos SQL.

4. Anidación de datos: Los datos contienen estructuras anidadas (como types que 
   es una lista de objetos, o stats que también es una lista de objetos).

5. Flexibilidad: Permite que algunos campos puedan estar presentes o ausentes 
   sin romper la estructura general.

Este formato es ideal para APIs web porque mantiene la jerarquía de los datos 
y permite una fácil serialización/deserialización entre diferentes lenguajes 
de programación.

4.4 DESCRIPCIÓN DEL HISTOGRAMA
-------------------------------
La distribución de total-stats muestra las siguientes características:

1. Forma general: La distribución tiene una forma aproximadamente normal 
   (campana de Gauss) con una ligera asimetría hacia la derecha.

2. Rango de valores: Los stats totales varían desde aproximadamente 175 hasta 680, 
   mostrando una gran variabilidad en el poder general de los pokémones.

3. Concentración: La mayoría de los pokémones se concentran en el rango medio 
   (alrededor de 400-500 stats totales), lo que indica que la mayoría tiene 
   un poder equilibrado.

4. Outliers: Existen algunos pokémones con stats totales muy altos 
   (legendarios como Mewtwo) y otros con stats muy bajos 
   (pokémones de evolución temprana).

5. Interpretación: Esta distribución sugiere que el juego está diseñado para 
   mantener un balance general, con la mayoría de pokémones teniendo capacidades 
   similares, mientras que algunos pocos destacan por ser excepcionalmente 
   fuertes o débiles.
